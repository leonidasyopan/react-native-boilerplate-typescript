# Boilerplate description

*Install:* run the following command to install dependencies:
```
yarn
```

This boilerplate has some of my favororite (and most times used) dependencies:
- eslint
- prettier
- typescript

*Run:* you need the emulator to be runnig, then type:
```
yarn android
```
